SetLeftMargin

SetLeftMargin(float margin)

Version

1.4

Beschreibung

Legt den linken Randabstand fest. Diese Methode kann vor dem Erstellen der ersten Seite aufgerufen werden.
Wenn die horizontale Position die Seitenbreite übersteigt, wird die Einstellung auf den Randbereich zurückgesetzt.

Parameter

margin
Der Rand (entsprechend der festgelegten Maßeinheit).
