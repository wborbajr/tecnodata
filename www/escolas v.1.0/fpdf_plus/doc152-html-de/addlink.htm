AddLink

int AddLink()

Version

1.5

Beschreibung

Erstellt eine neue interne Verknüpfung und gibt den Kennzeichner der Verknüpfung zurück. Eine interne Verknüpfung ist ein anklickbarer Bereich, welcher auf eine andere Position im selben Dokument verweist.
Der Kennzeichner kann an Cell(), Write(), Image() oder Link() übergeben werden. Die Sprungmarke (das Verweisziel) wird mit SetLink() definiert.
