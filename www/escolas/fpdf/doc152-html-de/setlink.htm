SetLink

SetLink(int link [, float y [, int page]])

Version

1.5

Beschreibung

Definiert die Seite und die Position, auf die eine Verknüpfung verweist.

Parameter

link
Der von AddLink() zurückgegebene Kennzeichner der Verknüpfung.
y
Die vertikale Position der Sprungmarke; -1 definiert die aktuelle Position. Der Standardwert ist 0 (Seitenanfang).
page
Die Seitenzahl; -1 definiert die aktuelle Seite (Standardwert).
